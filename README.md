# Milestone admin docs

A documentation site made using [VuePress](https://vuejs.press).

## Prerequisite

### Yarn

Install using:

```sh
npm install -g yarn
```

## Setup

```sh
yarn install
```

## Building

### Development server

```
yarn docs:dev
```

Supposed to have filesystem watcher and automatically reloads when there are changes, but I cannot get it to work.

### Building the site

```
yarn docs:build
```
